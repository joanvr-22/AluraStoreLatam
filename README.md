# AluraStoreLatam
challenge 1

Este proyecto tiene como objetivo analizar el desempeño de las cuatro tiendas del Sr. Juan para identificar cuál presenta el menor rendimiento y así tomar una decisión informada sobre cuál vender para invertir en un nuevo negocio con mayor potencial de crecimiento. Para realizar este análisis se evaluaron diferentes indicadores, entre ellos los ingresos totales por tienda, las categorías de productos más vendidas, los productos con mayor número de ventas, las calificaciones promedio de los clientes y los costos promedio de envío.

El proyecto se organiza en una estructura sencilla que permite mantener los datos, el análisis y la documentación ordenados. Dentro del proyecto se encuentra el archivo AluraStoreLatam.ipynb con los urls que contiene los datos utilizados para el análisis y todo el procesamiento y visualización de la información.

Durante el análisis se utilizaron distintas visualizaciones para facilitar la interpretación de los datos y obtener insights relevantes. Se utilizó un gráfico de área de densidad para analizar la distribución de los ingresos entre las tiendas, una gráfica de pastel para identificar las categorías de productos más vendidas, gráficas de barras horizontales para visualizar los productos con mayor número de ventas, y gráficas de barras para comparar tanto las calificaciones promedio de los clientes como el costo promedio de envío entre las tiendas. Estas visualizaciones permitieron identificar patrones de comportamiento en las ventas y el desempeño general de cada tienda.

A partir de los resultados obtenidos, se observó que la Tienda 4 presenta los ingresos totales más bajos en comparación con las demás tiendas, lo que indica un menor desempeño comercial general. Aunque su costo promedio de envío es menor, esta ventaja no logra compensar el nivel inferior de ingresos. Por esta razón, el análisis sugiere que la Tienda 4 sería la mejor candidata para vender, permitiendo que el Sr. Juan pueda utilizar esos recursos para invertir en un nuevo negocio con mayores oportunidades de crecimiento.

Para ejecutar el proyecto es necesario contar con Python. Una vez instaladas, se debe abrir Jupyter Notebook y ejecutar el archivo AluraStoreLatam.ipynb, donde se encuentran todos los pasos del análisis y las visualizaciones utilizadas para llegar a las conclusiones del estudio.
